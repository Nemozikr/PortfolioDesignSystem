Text Colours

Black #000

Description: This is going to be the default colour for the majority of the text to reach a good colour combination and a good contrast.

Greenish #85B44B

Description: This is a colour which is paired with the "Purplish" colour to form a contrast and adds colour to the page.

Background Colours

White #FFF

Description: Used for max contrast and as a default black and white colour combination.

Purplish #7A4BB4

Description: This colour is paired with the "Greenish" colour to create contrast and make webpage more interesting with colour variation.

Text Fonts

Roboto Condensed

  • Boldness: 500
  • Style: Normal

Description: Default font for the titles and headers

Sample: Pellentesque habitant morbi tristique senectus et netus et malesuada fames ac turpis egestas.

Arial, Helvetica, sans-serif

  • Boldness: A variety, higher boldness for headers.
  • Style: Normal for normal text, italic for supporting text.

Description: Default font for paragraphs, normal text etc.

Sample: Pellentesque habitant morbi tristique senectus et netus et malesuada fames ac turpis egestas.
